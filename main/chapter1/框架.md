# 基于人体几何先验的可驱动穿衣人重建研究

## 问题挑战
* 高斯与网格拓扑解耦 → 分布外姿态下表面破碎与伪影
* 服装生成缺乏体型感知 → 版型失配且依赖大量训练数据

## 研究思路
引入显式几何先验：网格拓扑先验（外观重建） + 人体测量先验（服装生成）

## 研究内容
1. 拓扑感知的三维高斯人体重建（第三章）
    * 可变形穿衣人网格：SMPL + 神经形状编码器 → 个性化网格 → LBS 驱动
    * 解析式高斯绑定：面片高斯（内切圆锚点 + 面法向推导）+ 顶点高斯（顶点中心 + 一环边长约束），几何属性全部解析推导
    * 等边几何正则化：约束三角面片趋向等边，维持拓扑质量
2. 基于人体几何先验的服装参数生成（第四章）
    * 人体测量提取：GarmentMeasurements 适配 SMPL → 单图提取 26 项体表测量值
    * 设计参数重表示：信息分流压缩 886 行 → ≤40 行紧凑格式
    * 几何先验驱动推理：锚点归一化 + 分步工作流 → GPT-4o 零样本多模态推理

## 目标
* 提升外观重建几何质量与驱动泛化性
* 提升服装参数生成精度与推理效率
