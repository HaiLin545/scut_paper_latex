\section{研究背景与意义}

构建逼真的穿衣人体数字模型是计算机视觉与计算机图形学中的基础性问题，在虚拟试衣、远程通信、影视特效和医疗康复等领域有着广泛的应用需求。一个完整的穿衣人体数字化方案需要同时解决两方面的问题：一是准确重建人体的形状、姿态与表面外观，使数字人能够在不同姿态下保持几何完整性；二是对人体所穿着的服装进行建模，使生成的服装在款式、尺寸和穿着效果上与真实观测一致。传统的人体数字化方案依赖多相机阵列或结构光扫描仪等专业设备，虽然能够获取高精度的几何模型，但设备成本高、场景限制大，难以满足消费级应用的需求。随着移动终端的普及和虚拟交互场景的增长，面向单目图像或视频的轻量化穿衣人体建模技术日益受到关注。


参数化人体模型是上述研究的共同基础。以SMPL\cite{smpl}为代表的参数化模型通过低维的体型参数和姿态参数控制人体网格的形状与运动，为从图像中恢复人体三维结构提供了有效的先验约束。一方面，SMPL的固定拓扑结构可以作为高斯基元、纹理贴图等表面表示的载体，支撑可驱动人体的重建；另一方面，SMPL所编码的人体几何信息（如臂长、腰围、腿长等）可以作为身体测量先验，为服装的尺寸推算提供物理参照。然而，SMPL模型本身仅能表达裸体形状的低频变化，无法建模衣物、发型等高频几何细节，这要求在其基础上引入更具表达能力的表示技术。

在三维人体重建方面，近年来涌现了多种基于学习的表示方法。神经辐射场（NeRF）\cite{nerf}通过隐式连续函数编码场景的几何与外观，实现了高质量的新视角合成，但其逐射线的体渲染机制导致训练和推理速度难以满足实时交互的需求。三维高斯泼溅（3DGS）\cite{3dgs}作为近年来兴起的显式表示技术，以一组各向异性的三维高斯基元作为场景建模单元，通过可微光栅化实现了实时渲染与高质量重建的兼顾，已迅速成为动态人体重建的主流技术路线。将3DGS与SMPL结合用于可驱动人体建模时，现有方法\cite{gaussian-avatar, gart, gom-avatar, ihuman}普遍将高斯基元绑定至SMPL网格上，通过线性混合蒙皮驱动高斯基元的空间运动。但这些方法通常将高斯基元的旋转和缩放属性作为自由可学习参数进行无约束优化，使得高斯属性与底层网格拓扑之间缺乏显式关联。其后果是：模型在训练姿态下能取得较好的渲染效果，但当人体进入训练集中未出现过的姿态时，高斯基元因无法感知网格表面的局部拉伸与压缩而脱离网格表面，产生表面空洞与几何伪影，严重限制了可驱动数字人在实际交互场景中的应用。

在服装建模方面，上述基于SMPL网格的重建方案适用于贴身衣物（如紧身上衣、紧身裤等），因为这类衣物的形变模式与人体表面高度一致，可以通过学习SMPL网格的顶点偏移量来近似表达。然而，对于非贴身服装（如裙摆、宽松外套、多层叠穿等），衣物具有独立于人体的运动模式和拓扑结构，SMPL的固定网格无法对其进行有效表达——例如，抬腿时裙摆会自然分开，但SMPL网格中双腿间的面片始终相连，导致重建结果出现拓扑分裂。为突破这一局限，一条可行的思路是将研究视角从"在人体表面上重建服装外观"转向"服装几何的参数化生成"：先从图像中恢复结构化的服装设计参数，再借助参数化制版系统自动生成二维缝纫版型与三维服装几何。GarmentCode\cite{GarmentCode2023}等参数化制版框架的出现为上述思路提供了技术基础。然而，现有的图像到服装参数生成方法\cite{zhou2025design2garmentcode}主要依赖多模态大语言模型进行视觉参数预测，其回归过程缺乏对穿着者真实体型的显式感知——单凭图像难以准确判断"袖长与臂长的比例"或"裙长与腿长的比例"等依赖于身体尺寸的空间关系，导致生成的服装版型在绝对尺寸上与穿着者的体型不匹配。

综上所述，围绕穿衣人体的数字化建模，当前研究面临两个核心问题：（1）基于3DGS的可驱动人体重建中，高斯基元的几何属性与网格拓扑之间缺乏显式关联，导致在训练分布外的姿态下出现几何不一致；（2）从图像生成服装参数时，缺乏对穿着者人体几何的显式感知，导致服装尺寸与体型失配。这两个问题分别对应穿衣人体建模的两个层面——人体表面的高保真重建与服装几何的精确生成，且都可以从参数化人体模型的信息中找到解决线索。

针对上述问题，本文以SMPL参数化人体模型为共同的几何基础，分别从人体表面的拓扑感知重建和基于人体先验的服装参数化生成两个方面展开研究。第一项工作面向可驱动人体重建，提出将高斯基元的旋转与缩放由SMPL网格的局部几何特征解析确定，使高斯属性能够随网格形变自适应调整，从而在任意姿态下保持几何一致性。第二项工作面向服装参数化生成，提出从SMPL模型中提取人体测量参数作为几何先验，注入多模态大语言模型的推理过程中，使服装参数的生成能够在显式的身体尺寸约束下完成，从而保证生成的服装版型与穿着者体型相匹配。两项工作形成了自然的递进关系：第一项工作在SMPL的固定拓扑上实现了对贴身穿衣人体的高保真重建，但该方案面对非贴身服装时存在拓扑表达的局限；第二项工作从另一个角度出发，通过参数化制版框架将服装建模从SMPL的固定网格中解放出来，转而利用SMPL提供的人体测量先验来约束服装参数的生成，弥补了第一项工作在服装类型上的限制。两项工作以SMPL参数化人体模型为纽带，共同推进了从人体表面重建到穿衣人体完整建模的技术路径。

\section{研究内容与贡献}

本文围绕穿衣人体的三维重建与服装参数化生成展开研究，整体研究框架如图\ref{fig:ch1-framework}所示,主要包含以下两方面内容：


\noindent\textbf{（一）基于拓扑感知的三维高斯人体重建方法（第三章）}

针对现有基于3DGS的可驱动人体方法中高斯属性与网格拓扑解耦导致的姿态泛化能力不足问题，本文提出了一种拓扑感知的三维高斯人体重建方法。该方法的核心思想是将高斯基元的旋转与缩放从可学习的自由参数转变为由网格局部几何特征解析确定的确定性函数。具体而言，本文以可变形SMPL网格为拓扑骨架，通过神经形状编码器学习顶点偏移量以捕捉个性化衣物细节，并在网格面片和顶点处分别构建两类互补的高斯基元：面片高斯利用内切圆半径确定缩放、利用面片法向量构建旋转，提供结构化的表面覆盖；顶点高斯利用邻域最短边长确定缩放、利用顶点法向量构建旋转，刻画高频细节并改善面片交界处的连续性。此外，本文还提出等边正则化约束，防止网格三角形在关节弯曲处退化为狭长形状，保障解析推导的数值稳定性。在PeopleSnapshot\cite{people-snapshot}和X-Avatar\cite{x-avatar}两个公开数据集上的实验表明，该方法在保持高质量渲染效果的同时，法向误差相比最优基线分别降低17.9\%和13.8\%，高斯数量减少62.5\%，模型大小降低44.5\%。

本项工作的主要贡献包括：
\begin{enumerate}
    \item 提出了拓扑感知的高斯属性解析推导机制，将高斯基元的旋转与缩放由网格局部微分几何性质（法向量、内切圆半径、边长）解析确定，在原理上保证了高斯基元与人体表面在任意姿态下的拓扑一致性。
    \item 设计了面片高斯与顶点高斯相结合的双层高斯表示结构，面片高斯提供粗粒度的表面覆盖，顶点高斯补充高频细节与面片交界处的连续性，两者协同实现了完整且精细的人体表面建模。
    \item 提出了等边正则化项，通过约束三角形的边长均匀性和内角稳定性，防止网格面片在姿态驱动过程中退化，保障解析推导框架在大幅度姿态变化下的数值稳定性。
\end{enumerate}

\noindent\textbf{（二）基于人体几何先验的服装参数生成方法（第四章）}

针对现有图像到服装参数生成方法中缺乏人体几何感知、服装版型与穿着者体型失配的问题，本文提出了一种基于人体几何先验的服装参数生成方法。该方法的核心思想是将SMPL参数化人体模型所编码的身体几何信息作为显式先验引入服装参数的推理过程。具体而言，本方法首先利用单目人体重建技术估计穿着者的SMPL模型参数，并通过身体测量工具自动提取语义明确的测量数据（如臂长、腰围、腿长等）；随后将身体测量数据与GarmentCode参数规范共同编码为多模态大语言模型（GPT-4o）的推理上下文，使模型能够在显式的身体尺寸约束下完成视觉分析与参数生成；最后通过格式转换模块将模型输出转化为GarmentCode可执行的设计参数，驱动制版系统自动生成二维缝纫版型与三维服装几何。该方法无需针对特定服装品类进行训练或微调，具备零样本泛化能力。

本项工作的主要贡献包括：
\begin{enumerate}
    \item 提出了融合人体几何先验的服装参数生成框架，通过从SMPL模型中提取身体测量参数并注入大语言模型的推理上下文，为视觉参数预测提供了显式的空间尺度参照，构建了从单幅图像到可执行设计参数的完整自动化链路。
    \item 设计了面向GarmentCode的规范化提示机制，通过"生成规范+几何先验+输出格式"的多重约束，使通用的多模态大语言模型无需额外训练即可输出符合领域规范的结构化设计参数。
    \item 完成了SMPL模型网格在身体测量流程中的工程适配，打通了从三维人体重建到二维/三维服装制版的完整系统链路。
\end{enumerate}


\section{论文结构安排}

本文共分为五章，各章内容安排如下：

\textbf{第一章：绪论。}阐述本文的研究背景与意义，分析穿衣人体建模面临的核心技术挑战，概述本文的研究内容与主要贡献，并介绍论文的整体结构。

\textbf{第二章：相关研究综述。}围绕三维人体重建，依次综述参数化人体模型、基于神经辐射场的方法以及基于三维高斯泼溅的方法，分析现有方法在可驱动数字人建模中存在的拓扑一致性问题。围绕三维服装建模与生成，回顾服装的表示方法、基于几何重建的建模方法、基于参数化样板的生成方法以及多模态大模型驱动的生成方法，分析现有方法在人体几何感知方面的不足。

\textbf{第三章：基于拓扑感知的三维高斯人体重建方法。}详细阐述本文的第一项工作，包括可变形穿衣人网格的构建、面片高斯与顶点高斯的解析推导过程、等边正则化约束的设计，以及在PeopleSnapshot和X-Avatar数据集上的系统实验与分析。

\textbf{第四章：基于人体几何先验的服装参数生成方法。}详细阐述本文的第二项工作，包括人体参数估计与身体测量提取、规范化提示构建、多模态参数推理以及参数转换与下游生成机制，以及相应的实验评估。

\textbf{第五章：总结与展望。}总结全文的研究成果与贡献，讨论现有方法的局限性，展望未来的研究方向。
